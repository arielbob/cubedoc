# CubeDoc

CubeDoc is a website for speedcubing tutorials and tips.

![image](https://user-images.githubusercontent.com/23489769/51445697-32a89980-1cd6-11e9-8d18-3c7c4952b196.png)

## Installation

Install using [npm](https://www.npmjs.com/):
```bash
npm install
```

## Running

```bash
npm start
```

## Building

```bash
npm run build
```
